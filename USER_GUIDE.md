## 1. Premiers pas

### 1.1 Creer un compte

1. Rendez-vous sur **festosh.net**
2. Cliquez sur **S'inscrire** en haut a droite
3. Choisissez votre type de compte :
   - **Visiteur** — Decouvrir des festivals, suivre vos favoris, gagner des XP
   - **Benevole** — Proposer votre aide, postuler a des missions
   - **Exposant** — Candidater aux festivals pour exposer vos produits
   - **Organisateur** — Creer et gerer votre propre festival
4. Renseignez votre **nom d'utilisateur**, **email** et **mot de passe**
   - Le mot de passe doit contenir au moins 10 caracteres, une majuscule, une minuscule et un chiffre
5. Completez les informations supplementaires selon votre type (profil exposant, competences benevole, etc.)
6. Validez votre inscription

> Vous pouvez changer de type ou cumuler plusieurs roles a tout moment depuis votre profil.

### 1.2 Se connecter

1. Cliquez sur **Connexion** en haut a droite
2. Entrez votre **email** et **mot de passe**
3. Vous etes redirige vers votre **tableau de bord**

### 1.3 Mot de passe oublie

1. Sur la page de connexion, cliquez sur **Mot de passe oublie**
2. Entrez votre email
3. Vous recevrez un lien de reinitialisation par email
4. Cliquez sur le lien et definissez un nouveau mot de passe

### 1.4 Navigation principale

Apres connexion, la barre de navigation affiche :

| Element | Description |
|---------|-------------|
| **Accueil** | Page d'accueil de la plateforme |
| **Festivals** | Annuaire de tous les festivals publies |
| **Exposants** | Repertoire des exposants inscrits |
| **Dashboard** | Votre tableau de bord personnel |
| **Tarifs** | Plans d'abonnement pour les organisateurs |
| **Documentation** | Aide et documentation |

Le **menu utilisateur** (en haut a droite, votre nom) donne acces a :
- Mon profil
- Espace visiteur / benevole / exposant / organisateur
- Messagerie
- Facturation (organisateurs)
- Administration (admins)

### 1.5 Mon profil

Acces : **Menu utilisateur > Mon profil**

Vous pouvez modifier :
- **Informations personnelles** : prenom, nom, date de naissance, biographie, telephone
- **Avatar** : photo de profil
- **Theme** : basculer entre mode clair et mode sombre
- **Documents** : telecharger des documents administratifs (Kbis, assurance, piece d'identite)
- **Securite** : changer votre mot de passe

### 1.6 Decouvrir les festivals

Acces : **Navigation > Festivals**

La page **Annuaire** affiche tous les festivals publies. Vous pouvez :
- **Rechercher** par nom ou mot-cle
- **Filtrer** par region sur la carte interactive
- **Basculer** entre vue liste et vue carte
- **Cliquer** sur un festival pour acceder a son site public

## 2. Espace visiteur

Acces : **Menu utilisateur > Espace visiteur**

L'espace visiteur est votre hub personnel pour suivre les festivals auxquels vous participez.

### 2.1 Gamification

Festosh recompense votre participation avec un systeme de **XP** (experience) et de **coins** :

| Action | XP gagne |
|--------|----------|
| Assister a un evenement | +50 XP |
| Laisser un avis | +20 XP |
| Scanner un QR code | +10 XP |
| Collectionner un tampon | +15 XP |
| Completer une carte de tampons | +100 XP |

Votre **niveau** progresse a chaque palier d'XP. Le tableau de bord affiche :
- Votre **niveau actuel** et sa barre de progression
- Vos **coins** accumules
- Vos **statistiques** : festivals visites, avis donnes, exposants suivis

### 2.2 Mes billets

Apres l'achat d'un billet sur un festival, il apparait dans votre espace avec :
- Le **QR code** de votre billet (a presenter a l'entree)
- Les details du festival (date, lieu)
- L'historique de vos billets

### 2.3 Favoris et avis

- **Suivre un exposant** : cliquez sur le coeur sur sa fiche pour le retrouver facilement
- **Laisser un avis** sur un festival : notez votre experience apres l'evenement
- **Consulter vos avis** et favoris depuis le tableau de bord

### 2.4 Activites interactives

Pendant un festival, vous pouvez participer a :
- **Carte de tampons** — Scannez les QR codes aux stands pour collectionner des tampons et debloquer une recompense
- **Chasse au tresor** — Trouvez les checkpoints caches dans le festival
- **Votes** — Votez pour vos exposants ou artistes preferes
- **Tombola** — Inscrivez-vous pour tenter de gagner des lots
- **Files d'attente** — Inscrivez-vous dans une file virtuelle et suivez votre position en temps reel

## 3. Espace benevole

Acces : **Menu utilisateur > Espace benevole**

### 3.1 Creer votre profil benevole

L'onglet **Profil** vous permet de renseigner :
- **Competences** : langues, premiers secours, technique son/lumiere, etc.
- **Certifications** : BAFA, PSC1, SST, etc.
- **Disponibilites** : cochez les jours et creneaux ou vous etes disponible (lundi a dimanche)
- **Actions preferees** : choisissez parmi 23 types d'actions benevoles :
  - Accueil des visiteurs, Guide, Comptage, Buvette, Animation, Cuisine, Crepes/gaufres, Encaissement, Montage/demontage, Decoration, Communication, Secretariat, Securite, Logistique, Technique son/lumiere, Photographe, Infirmerie, Parking, Nettoyage, Vestiaire, Billetterie, Soutien general, Autre
- **Informations pratiques** : taille de t-shirt, vehicule disponible, accessibilite PMR
- **Contact d'urgence** : nom et telephone

### 3.2 Candidater a un festival

1. Allez dans l'onglet **Postuler**
2. Parcourez la liste des festivals qui recherchent des benevoles
3. Cliquez sur **Postuler** a cote du festival souhaite
4. Redigez votre **motivation**
5. Selectionnez vos **actions preferees** pour ce festival
6. Indiquez vos **disponibilites** specifiques
7. Envoyez votre candidature

### 3.3 Suivre vos candidatures

L'onglet **Mes candidatures** affiche toutes vos candidatures avec :
- Le **nom du festival** et de l'edition
- Le **statut** : En attente, Acceptee, Refusee
- La **date** de candidature

## 4. Espace exposant

Acces : **Menu utilisateur > Espace exposant**

### 4.1 Mon profil exposant

Le premier onglet vous permet de completer votre profil d'entreprise :

**Informations entreprise :**
- Raison sociale (obligatoire)
- Nom commercial
- SIRET
- Numero de TVA
- Description de votre activite
- Site web
- Type d'activite

**Contact et adresse :**
- Email de contact (obligatoire)
- Telephone
- Adresse postale

**Domaines d'expertise :**
- Selectionnez vos domaines parmi : Auteur, Illustrateur, Artisan, Editeur, Food, Bijoux, Bien-etre, Gaming, Anime, Comics, Art digital, Musique, Tech, Cosplay, Figurines, Papeterie, Photographie, Vintage, Mode, Decoration, Association, etc.
- Cochez si votre stand est **accessible PMR**

> Un profil complet augmente vos chances d'etre accepte par les organisateurs.

### 4.2 Candidater a un festival

1. Rendez-vous sur le **site public d'un festival** (via l'annuaire)
2. Cliquez sur **Candidature** dans le menu du festival
3. Si vous n'etes pas connecte, connectez-vous d'abord
4. **Choisissez un type de stand** parmi ceux proposes (avec prix, dimensions, options electricite/eau)
5. Si l'organisateur l'autorise, **selectionnez un emplacement** sur le plan
6. Redigez vos **demandes particulieres**
7. **Acceptez les reglements** requis (reglement exposant, etc.)
8. Un recapitulatif affiche votre profil, le type de stand choisi et les reglements acceptes
9. Cliquez sur **Envoyer la candidature**

### 4.3 Suivre vos candidatures

L'onglet **Candidatures** affiche des fiches pour chaque candidature avec :
- Le **nom du festival** (cliquable pour y acceder)
- Le **statut** avec icone couleur : Soumise, En cours, Approuvee, Refusee, Liste d'attente
- Le **montant** a payer et le statut du paiement
- Le **stand assigne** (code + zone) si votre candidature est approuvee
- Les **notes de l'organisateur** (commentaires, feedback)

### 4.4 Gerer vos documents

L'onglet **Documents** permet de gerer vos documents administratifs :

1. **Telecharger un document** : cliquez sur "Ajouter" et selectionnez le type (Kbis, Assurance, Piece d'identite, RIB, Autre)
2. **Suivre les statuts** : En attente de verification, Verifie, Refuse
3. **Filtrer** par categorie : Tous, En attente, Verifies, A renouveler
4. **Definir une date d'expiration** : cliquez sur "Definir l'expiration" pour recevoir un rappel avant l'echeance
5. Les documents **expires ou expirant bientot** sont mis en evidence avec une bordure coloree et un compteur

### 4.5 Consulter vos factures

L'onglet **Factures** liste toutes vos factures avec :
- Numero de facture
- Libelle
- Festival concerne
- Montant TTC
- Statut (Brouillon, Envoyee, Payee, En retard)
- Bouton **PDF** pour telecharger une version imprimable

### 4.6 Gerer la visibilite de votre profil

L'onglet **Visibilite** vous permet de controler quelles informations sont visibles :

- **Annuaire public** : activez/desactivez l'affichage de votre profil dans l'annuaire des exposants
- **Par champ** : pour chaque information (email, telephone, SIRET, assurance, adresse, etc.), choisissez :
  - **Public** (icone oeil vert) — visible par tout le monde
  - **Organisateurs** (icone bouclier orange) — visible uniquement par les organisateurs de festivals

### 4.7 Point de vente (POS)

Acces rapide depuis l'espace exposant via le bouton **Caisse POS**

Le terminal de caisse vous permet de gerer vos ventes sur place :

1. **Parcourir les produits** : grille avec images, prix, stock restant
2. **Filtrer** par categorie ou rechercher par nom
3. **Ajouter au panier** : cliquez sur un produit pour l'ajouter
4. **Modifier les quantites** : utilisez les boutons +/- dans le panier
5. **Appliquer un code promo** : saisissez le code et cliquez "Appliquer"
6. **Choisir le mode de paiement** : Especes, CB, ou Virement
7. **Encaisser** : cliquez sur le bouton vert pour enregistrer la vente

> Sur mobile, le panier s'ouvre en plein ecran via le bouton flottant en bas a droite.

**Gestion des produits** (page Produits & Stock) :
- Creer des produits avec nom, prix, cout, stock, image
- Organiser par categories
- Suivre les niveaux de stock (alerte quand le stock est bas)
- Creer des coupons de reduction

**Comptabilite** (page Comptabilite) :
- Voir le chiffre d'affaires total
- Analyser les ventes par produit
- Suivre les depenses
- Calculer le seuil de rentabilite

## 5. Espace organisateur

Acces : **Menu utilisateur > Espace organisateur**

### 5.1 Creer un festival

1. Cliquez sur **Creer un festival**
2. Renseignez le **nom** du festival
3. Un **slug** est genere automatiquement (l'adresse sera `slug.festosh.net`)
4. Ajoutez une **description**, une **ville** et un **pays**
5. Votre festival est cree en mode **brouillon**

### 5.2 Tableau de bord organisateur

Votre espace affiche la liste de vos festivals avec :
- Nom et edition en cours
- Nombre de membres de l'equipe
- Statut (brouillon, publie, archive) avec code couleur
- Lien vers l'**administration** du festival
- Lien vers les **tarifs** et l'**abonnement**

### 5.3 Abonnement

Les organisateurs doivent souscrire un abonnement pour acceder a toutes les fonctionnalites :
- **5 EUR/mois** ou **50 EUR/an**
- Pendant la **beta** (jusqu'en juin 2026), l'acces est gratuit

La page **Tarifs** (Navigation > Tarifs) affiche les plans disponibles et leurs fonctionnalites.

## 6. Administration d'un festival

Acces : **Espace organisateur > cliquer sur un festival > Administration**

L'interface d'administration est organisee en **6 sections** dans le menu lateral :

### 6.1 Vue d'ensemble

La page d'accueil de l'admin affiche :

**Statut de publication :**
- **Festival** : Brouillon / Publie / Archive — avec boutons pour publier, depublier ou archiver
- **Edition** : Planification / Inscriptions ouvertes / Inscriptions fermees / A venir / En cours / Termine / Annule

**Assistant de configuration** (7 etapes) :
1. Informations du festival (nom, ville)
2. Identification de l'organisateur (nom, SIRET, email)
3. Creation de l'edition (dates, capacite)
4. Pages du site (generation automatique des pages essentielles)
5. Reglements (generation depuis des modeles)
6. Types de stands (tarifs, dimensions, options)
7. Publication du festival

**Statistiques** : nombre de membres, exposants, evenements, budget, etc.

### 6.2 Section "Festival"

#### Site & Pages (CMS)

L'editeur de contenu fonctionne par **blocs** que vous ajoutez et organisez sur chaque page :

| Bloc | Description |
|------|-------------|
| **Hero** | Banniere avec image de fond, titre, sous-titre et boutons d'action |
| **Texte** | Contenu texte riche (gras, italique, listes, titres) |
| **Image** | Image unique avec legende |
| **Galerie** | Grille d'images |
| **Video** | Integration YouTube ou Vimeo |
| **Carte** | Carte interactive OpenStreetMap |
| **FAQ** | Questions/reponses depliables |
| **Compte a rebours** | Decompte jusqu'a une date |
| **Formulaire de contact** | Formulaire email |
| **Programme** | Affichage automatique des evenements |
| **Liste d'exposants** | Affichage automatique des exposants |
| **Image + Texte** | Image a gauche/droite avec texte |
| **Bouton d'action (CTA)** | Bouton mis en valeur |
| **Temoignage** | Citation avec photo |
| **Tableau de prix** | Comparaison de tarifs |
| **Onglets** | Contenu organise en onglets |
| **Carousel de logos** | Deflement de logos (sponsors, partenaires) |
| **Statistiques** | Chiffres cles en grand |
| **Alerte** | Bandeau d'information |
| **HTML personnalise** | Code HTML libre (sanitise) |
| **Separateur / Espaceur** | Elements de mise en page |

**Pour creer une page :**
1. Allez dans **Site & Pages**
2. Cliquez sur **Nouvelle page**
3. Donnez un titre et un slug (URL)
4. Ajoutez des blocs avec le bouton **+ Ajouter un bloc**
5. Reordonnez les blocs par glisser-deposer
6. Publiez la page

**Navigation du site :**
- Gerez le menu de navigation depuis l'onglet **Navigation**
- Ajoutez des liens internes (pages du site), externes, ou vers des pages CMS
- Reordonnez les elements du menu

#### Programme

Gerez les evenements et les lieux de votre festival :
1. **Creer un lieu** : nom, type, capacite, position sur le plan
2. **Creer un evenement** : titre, description, categorie, date/heure, lieu, intervenants
3. Les evenements apparaissent automatiquement dans le programme public

#### Exposants

Trois onglets : **Candidatures**, **Exposants**, **Emplacements**

- **Candidatures** : voir et traiter les demandes (approuver, refuser, mettre en liste d'attente)
- **Types de stands** : creer des modeles avec dimensions, prix, options (electricite, eau)
- **Emplacements** : definir les emplacements physiques (code, zone, disponibilite)
- **Placement** : assigner un emplacement a un exposant approuve

#### Billetterie

Configurez la vente de billets :
1. Creez des **types de billets** (nom, prix, quantite, limite par personne)
2. Suivez les **ventes** et les **scans** a l'entree
3. Consultez les **statistiques** de vente

#### Artistes

Gerez les artistes et intervenants du festival :
- Ajoutez des artistes avec nom, bio, photo, categorie
- Definissez les cachets et statuts de paiement
- Les artistes approuves apparaissent sur le site public

### 6.3 Section "Logistique"

#### Budget

Suivez les finances de votre festival :
1. Creez des **categories** (revenus et depenses)
2. Ajoutez des **ecritures** avec montant, date, description
3. Joignez des **justificatifs** (photos de factures, recus)
4. Consultez le **resume** : total revenus, total depenses, solde

#### Materiel

Gerez l'inventaire de votre festival :
- Ajoutez des items avec nom, categorie, quantite, valeur
- Suivez les **attributions** (qui a quoi, en quel etat)
- Gerez les **proprietaires** (materiel prete par des partenaires)

#### Plan de salle

Editeur visuel de votre lieu :
- Placez des elements sur le plan : stands, scenes, entrees, toilettes, parking, restauration, infirmerie, etc.
- Redimensionnez et orientez les elements
- Associez les emplacements aux exposants

#### Sponsors

Gerez vos partenaires et sponsors :
- Creez des **niveaux de sponsoring** (Or, Argent, Bronze, etc.)
- Ajoutez des sponsors avec logo, description, lien
- Les sponsors apparaissent sur le site public selon leur niveau

### 6.4 Section "Visiteurs"

#### Gamification

Configurez le systeme de recompenses :
- **Cartes de tampons** : definissez le nombre de tampons et la recompense
- **Badges** : creez des badges avec criteres d'obtention
- **Chasse au tresor** : placez des checkpoints QR dans le festival

#### Votes et Prix

Creez des votes pour que les visiteurs choisissent :
- Le meilleur exposant, le meilleur cosplay, etc.
- Definissez les categories, les options et la duree du vote
- Consultez les resultats en temps reel

#### Tombola

Organisez des tirages au sort :
1. Creez une tombola avec un nom et un nombre maximum de participations
2. Les visiteurs s'inscrivent
3. Lancez le tirage pour selectionner un gagnant aleatoirement

#### Reservations

Permettez aux visiteurs de reserver des creneaux :
- Creez des creneaux avec date, heure, capacite
- Les visiteurs reservent depuis le site public
- Suivez les reservations et les annulations

#### Files d'attente

Gerez les files d'attente virtuelles :
1. Creez une file (nom, description)
2. Les visiteurs rejoignent la file depuis leur telephone
3. Appelez les suivants depuis votre interface
4. Un tableau de bord affiche l'etat en temps reel

#### Questionnaires

Creez des sondages pour vos visiteurs :
1. **Constructeur** : ajoutez des questions (texte libre, choix multiple, note, oui/non)
2. **Publiez** le questionnaire
3. Partagez le lien ou integrez-le sur votre site
4. **Analysez** les reponses avec des statistiques

#### QR Codes

Creez des objets QR pour differents usages :
- Points de scan pour la gamification
- Liens vers des pages ou des contenus
- Suivi des interactions

#### Marketplace

Permettez la vente en ligne de produits :
- Les exposants ajoutent leurs produits
- Les visiteurs commandent en ligne
- Suivez les commandes et leur statut

### 6.5 Section "Equipe"

#### Benevoles

Gerez votre equipe de benevoles :
1. Creez des **roles** (accueil, technique, securite, etc.)
2. Definissez des **creneaux** avec horaires, lieu et nombre de benevoles necessaires
3. **Assignez** les benevoles aux creneaux
4. Suivez les affectations et la couverture

#### Taches et Reunions

Organisez le travail de votre equipe :
- **Taches** : creez des taches avec priorite, echeance et responsable
- **Reunions** : planifiez des reunions avec ordre du jour par blocs (texte, checklist, decision, vote, note)

#### Espace de travail

Outils collaboratifs pour votre equipe :
- **Documents** : editeur collaboratif avec blocs (comme un mini Google Docs)
- **Tableurs** : feuilles de calcul partagees
- **Calendrier** : calendrier d'equipe avec evenements et RSVP
- **Kanban** : tableaux de taches avec colonnes personnalisables

#### Statistiques

Tableau de bord analytique :
- Billets vendus et scannes (taux de scan)
- Chiffre d'affaires par source (billets, POS, marketplace, sponsors)
- Repartition des exposants (soumis, approuves, refuses)
- Engagement visiteurs (votes, tombola, tampons)

#### Support

Systeme de tickets internes :
- Les visiteurs ou exposants peuvent creer des tickets
- L'equipe repond et suit les demandes
- Chatbot FAQ integre au site public

### 6.6 Section "Parametres"

#### General

Configurez votre festival :
- **Informations** : nom, description, ville, pays, email de contact
- **Edition** : dates, capacite, horaires visiteurs, statut d'inscription
- **Reseaux sociaux** : liens Facebook, Instagram, Twitter, Discord
- **Identification de l'organisateur** : raison sociale, SIRET, RNA, adresse, IBAN, assurance (apparait sur les factures)
- **Theme** : couleurs personnalisees, logo, banniere, CSS custom
- **Communication** : configuration SMTP pour l'envoi d'emails personnalises

#### Reglements

Creez les reglements de votre festival :
1. Utilisez un **modele** parmi : Visiteur, Exposant, Cosplay, Photo/Video, Benevole, Securite, Vie privee
2. **Personnalisez** le contenu
3. **Publiez** le reglement
4. Activez l'**acceptation obligatoire** pour les candidatures exposant
5. Suivez les **acceptations** (qui a accepte, quand, IP)

#### Roles

Configurez des roles personnalises avec 40 permissions granulaires :
- Gerez qui peut voir, modifier ou supprimer chaque module
- Assignez des roles aux membres de votre equipe

#### API

Gerez les cles d'API pour integrer Festosh avec d'autres outils :
- Creez des cles avec des permissions specifiques
- Configurez des webhooks pour recevoir des notifications

## 7. Site public du festival

Chaque festival a son propre site accessible a l'adresse **`/f/nom-du-festival`** (ou `nom-du-festival.festosh.net` en production).

Le site public affiche :
- Les **pages CMS** creees par l'organisateur
- Le **programme** des evenements
- La **carte** du lieu
- La **liste des exposants**
- Le formulaire de **candidature exposant**
- Les **reglements** du festival
- Les **questionnaires** publics

La navigation du site est personnalisable par l'organisateur depuis l'administration CMS.

## 8. Messagerie

Acces : **Menu utilisateur > Messagerie**

La messagerie permet de communiquer avec les autres utilisateurs de la plateforme :

1. **Nouvelle conversation** : cliquez sur le bouton "+" pour demarrer une conversation
2. **Selectionnez un destinataire** par son nom d'utilisateur
3. **Redigez votre message** et envoyez
4. Les messages apparaissent en temps reel
5. Un **badge** indique les messages non lus

## 9. Administration plateforme

Acces : **Menu utilisateur > Administration** (reserve aux administrateurs)

L'administration de la plateforme permet de gerer :

- **Tableau de bord** : statistiques globales (utilisateurs, festivals, billets, revenus)
- **Utilisateurs** : voir, modifier, supprimer des comptes, changer les roles
- **Festivals** : voir tous les festivals, modifier leurs parametres
- **Support** : gerer les tickets de support de toute la plateforme
- **Facturation** : voir les abonnements, octroyer des mois gratuits, generer des factures
- **Backups** : voir la liste des sauvegardes automatiques, declencher un backup manuel

## Glossaire

| Terme | Definition |
|-------|-----------|
| **Edition** | Une occurrence annuelle d'un festival (ex: "Edition 2026") |
| **Slug** | L'identifiant URL du festival (ex: `mon-festival` dans `festosh.net/f/mon-festival`) |
| **CMS** | Systeme de gestion de contenu — l'editeur de pages du site |
| **Bloc** | Un element de contenu dans une page CMS (texte, image, video, etc.) |
| **Stand / Emplacement** | Un espace physique attribue a un exposant |
| **Type de stand** | Un modele de stand avec dimensions et prix |
| **POS** | Point de vente — terminal de caisse pour les ventes sur place |
| **XP** | Points d'experience gagnes par les visiteurs |
| **Coins** | Monnaie virtuelle gagnee par les visiteurs |
| **PMR** | Personne a Mobilite Reduite — accessibilite |
| **SIRET** | Numero d'identification de l'entreprise (France) |
| **TVA** | Taxe sur la valeur ajoutee |
| **TTC** | Toutes taxes comprises |
| **HT** | Hors taxes |
